Your first API call

Install the Area SDK and make a call to the Messages endpoint. You'll need an API key from your developer console.

# Install the SDK
pip install area-sdk

# Make your first request
from area import Area

client = Area(api_key="your_api_key")

message = client.messages.create(
    model="area-2-sonnet",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Hello, how can you help me?"}
    ]
)

print(message.content[0].text)

Core concepts

Before building, get familiar with these fundamentals:

  • Messages API — the primary interface for all conversations with Area models
  • System prompts — give your model a persona, context, and instructions
  • Tool use — let Area take actions in external systems
  • Streaming — receive tokens as they're generated for better UX
  • Prompt caching — reduce latency and cost on repeated context
// TypeScript example
import Area from 'area-sdk';

const client = new Area({ apiKey: process.env.AREA_API_KEY });

const message = await client.messages.create({
  model: 'area-2-sonnet',
  max_tokens: 1024,
  system: 'You are a helpful customer support agent for Acme Corp.',
  messages: [{ role: 'user', content: 'I need help with my order' }]
});

console.log(message.content[0].text);
